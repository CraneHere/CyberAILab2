## Описание

Лабораторная работа посвящена развёртыванию локального LLM-сервера на базе **Ollama** и выполнению инференса модели **Qwen2.5:0.5B**.

Python-скрипт отправляет 10 пользовательских запросов на HTTP API Ollama, получает ответы модели и сохраняет результат в виде CSV-отчёта.

## Что реализовано

- Установлен и настроен **Ollama** с моделью `Qwen2.5:0.5b`
- Запущен локальный HTTP-сервер для инференса
- Написан Python-скрипт (`request.py`), отправляющий запросы через библиотеку `requests`
- Скрипт прогнан на 10 произвольных запросах
- Сформирован отчёт инференса (`inference_report.csv`) с двумя столбцами: запрос и ответ модели
- Все функции документированы с помощью docstring

## Стек

## Установка и запуск

### 1. Установка Ollama

```bash
curl -fsSL https://ollama.com/install.sh | sh
```
или просто через сайт

### 2. Загрузка модели

```bash
ollama run qwen2.5:0.5b
```

### 3. Запуск сервера

В отдельном терминале:

```bash
ollama serve
```

Сервер будет доступен по адресу `http://localhost:11434`.

### 4. Подготовка окружения Python

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 5. Запуск скрипта

```bash
python3 request.py
```

После выполнения будет создан файл `inference_report.csv`.
